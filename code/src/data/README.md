# Data Directory

This folder contains static assets, mock data for testing, and pre-defined layout templates.

## Contents
* **`ElectionXXX.json`**: Mock election metadata used to simulate real-world voter environments.
* **`templates/`**: Standardized layout JSON files (e.g., Ballots, Notices) that can be loaded into the canvas.
* **`voterMailMergeSample...json`**: Sample datasets for testing the mail merge engine, covering both canonical and tabular formats.

# Test Directory

Automated test suite using Vitest and React Testing Library.

## Test Categories
* **Unit Tests**: Logic verification for services like `mailMergeService.test.ts` and `layoutService.test.ts`.
* **Integration Tests**: Validates complex workflows like `AppMailMergeUpload.test.tsx` and `CanvasInteraction.test.tsx`.
* **Hook Tests**: Specialized testing for stateful hooks like `usePageState.test.ts` and `useAppControllerDnd.test.ts`.
